# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 0.x (latest) | ✅ |

## Reporting a Vulnerability

セキュリティ上の脆弱性を発見した場合は、**GitHub の Public Issue には投稿しないでください**。

代わりに、以下のいずれかの方法でご連絡ください：

- **GitHub Private Security Advisory**: [Report a vulnerability](https://github.com/hirontan/gmo-coin-fx-rs/security/advisories/new)

報告内容には以下を含めてください：

1. 脆弱性の概要
2. 影響を受けるバージョン
3. 再現手順（可能な限り詳細に）
4. 想定される影響範囲

報告後、48時間以内に確認の返信をします。
